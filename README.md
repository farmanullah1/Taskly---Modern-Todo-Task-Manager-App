# 🌌 Taskly: Cosmic Productivity (Pro v4.0)

Taskly has been reborn as the world's most beautiful, hyper-modern task manager. Experience "Cosmic Glassmorphism" — a design language built with deep space gradients, heavy blur effects, and high-fidelity animations.

## 🌠 The Cosmic Experience

- **🌑 Cosmic Glassmorphism**: A stunning UI featuring a radial nebula background, high-blur cards, and glowing borders.
- **⚡ Focus Mode**: Enter deep-work mode where the UI strips away distractions to show you only one mission at a time.
- **🚀 Mission Hub**: A completely reimagined dashboard with a "Productivity Pulse" (real-time completion stats) and daily streaks.
- **🌈 Nebula Gradients**: Every element is infused with deep cosmic gradients (Violet → Pink → Cyan).
- **✨ Hyper-Animated**: Staggered list entries, magnetic hover effects, shimmering loading states, and spring-based transitions.
- **🛰️ Mission Control**: Expandable task input with due dates, priority levels, and detailed briefing notes.
- **📦 Instant Backup**: One-click export of your entire mission database to a JSON file.
- **📱 Universal Reach**: Fully responsive across mobile, tablet, and ultra-wide desktops.

## 🛠️ Tech Stack (The Engine)

- **Framework**: React 19
- **Design System**: Tailwind CSS v4 (Custom Cosmic Tokens)
- **Typography**: Outfit (Modern SaaS Font)
- **Motion**: Framer Motion (60 FPS fluid animations)
- **Physics**: canvas-confetti
- **Audio**: use-sound (Spatial UI feedback)
- **Icons**: Lucide Cosmic Set

## 🛠️ Local Development

1. **Clone & Install**:
   ```bash
   npm install
   ```
2. **Start Engines**:
   ```bash
   npm run dev
   ```
3. **Build Mission**:
   ```bash
   npm run build
   ```
