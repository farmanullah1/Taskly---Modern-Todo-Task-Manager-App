# 🚀 Taskly - The Modern Task Manager (Ultimate Edition)

Taskly is a premium, high-performance Todo application built with **React**, **Tailwind CSS**, and **Framer Motion**. It features smooth drag-and-drop reordering, persistent storage, and a stunning glassmorphism design.

[![Taskly Live](https://img.shields.io/badge/Live-Demo-blueviolet?style=for-the-badge&logo=vercel)](https://farmanullah1.github.io/Taskly---Modern-Todo-Task-Manager-App/)
[![Portfolio](https://img.shields.io/badge/My-Portfolio-blue?style=for-the-badge&logo=react)](https://farmanullah1.github.io/My-Portfolio/)

## ✨ Ultimate Features

- **🎨 Multi-Theme System**: Switch between various accent colors (Indigo, Rose, Emerald, Amber, Violet, Sky) with one click.
- **📊 Visual Analytics**: Real-time category statistics with circular progress charts and a global productivity bar.
- **🔍 Advanced Search**: Filter through tasks and notes instantly with real-time text matching.
- **📅 Smart Task Input**: Add tasks with due dates, priorities, and detailed notes using an expandable pro interface.
- **🎊 Celebration Engine**: High-priority tasks and full list completion trigger beautiful confetti bursts and spatial sound effects.
- **📥 Data Portability**: Export your entire task database to a JSON backup file.
- **⌨️ Power User Shortcuts**: Interactive keyboard shortcut guide for `Enter`, `Esc`, `Space`, and `Drag`.
- **📱 Fully Responsive**: Pixel-perfect layout from mobile phones to 4K displays with touch-optimized controls.
- **🌓 Adaptive Dark Mode**: System-aware or manual dark/light mode toggle with persistence.
- **🚀 SEO Optimized**: Fully optimized with Open Graph tags, meta descriptions, and semantic HTML for search visibility.

## 🛠️ Tech Stack

- **Framework**: React 19
- **Styling**: Tailwind CSS v4 (Custom Design System)
- **Animations**: Framer Motion
- **Drag & Drop**: @dnd-kit
- **Icons**: Lucide React
- **Celebrations**: canvas-confetti
- **Audio**: use-sound
- **Build Tool**: Vite

## 🚀 Getting Started

### Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:
```bash
npm run dev
```

### Build & Deploy

Create a production-ready build:
```bash
npm run build
```

Deploy to GitHub Pages:
```bash
npm run deploy
```
